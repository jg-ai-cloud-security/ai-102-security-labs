# AI-102 Day-3 — Core Azure AI Services

This lab covers hands-on deployment and testing of:

• Azure AI Language (PII, Key Phrases)
• Azure AI Vision (Face detection)
• Azure AI Document Intelligence (Invoice extraction)
• Responsible AI governance
• Service selection patterns
